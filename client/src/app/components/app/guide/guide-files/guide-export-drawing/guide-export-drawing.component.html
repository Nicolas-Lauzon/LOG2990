Exportation

L’outil exporter permet de sauvegarder un dessin localement ou de l’envoyer par courriel.

Il est possible d’exporter selon trois formats : PNG, JPG et SVG. Il est possible d’exporter les trois en même temps. Le dessin exporté doit avoir un titre composé de lettre et de chiffres seulement et ayant une longueur maximale de 50 caractères.

Il est possible d’appliquer un filtre sur l'image avant de l'exporter. L’application offre un choix de cinq filtres différents : Sombre, Saturation, Inversion de Couleur, Flou et Couleurs Froides. Un seul filtre doit pouvoir être appliqué à la fois.

À la droite du menu, une vignette de prévisualisation du dessin est présente pour montrer le dessin avec les modifications actives.

Exporter localement

Pour exporter localement, il faut s’assurer que la case "ENVOYER PAR COURRIEL" n’est pas active. Si elle n’est pas active, le dessin sera exporté localement.

Envoyer le dessin par courriel

Pour exporter un dessin et l’envoyer par courriel, il faut cocher la case "ENVOYER PAR COURRIEL" puis entrer une adresse valide dans la zone de texte qui apparaitra sous le bouton. Une fois l’adresse entrée, il suffit d’appuyer sur le bouton exporter dans le coin inférieur gauche du menu exporter.
