Palette de couleur

La palette de couleur est un outil permettant de choisir une couleur spécifique. Cette couleur peut être utilisé par les outils ou par le fonds du dessin.

La palette de couleur permet de sélectionner une couleur précise de deux façons différentes.

1. La première façon est de faire un clic gauche sur la bande de couleur RGB puis de choisir le gradient de cette couleur dans le rectangle de gauche de la palette de couleur.

2. La deuxième méthode est d’entrer le code hexadécimal du RGB de la couleur voulu dans les trois champs prévus à cet effet dans la partie droite droite de la palette de couleur.

La couleur choisie est ensuite montrée dans le carré dans le coin supérieur droit de la palette de couleur. Pour confirmer le changement de couleur, il faut peser sur le bouton "OK". Si on ne veut finalement pas changer la couleur, on peut cliquer à l’extérieur de la palette ou sur le bouton "ANNULER" pour fermer celle-ci.
